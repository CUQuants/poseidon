# Poseidon

Rust-based Kraken dashboard designed for CU Quants' trading team.

## Features
- View account balances in real-time
- Place, modify, and cancel orders
- Manage subaccounts
- Track order history
- High-performance, safe, and reliable

## Setup
1. Install Rust
2. Clone repository
3. Configure Kraken API keys
4. Run `cargo run`
